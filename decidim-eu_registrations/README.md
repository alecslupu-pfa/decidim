# Decidim::EuRegistrations

EU Registration.

## Usage

EuRegistrations will be available as a Component for a Participatory
Space.

## Installation

Add this line to your application's Gemfile:

```ruby
gem "decidim-eu_registrations"
```

And then execute:

```bash
bundle
```
